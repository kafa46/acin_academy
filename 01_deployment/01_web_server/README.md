# ACIN Academy에 방문하신 것을 환영합니다.

딥러닝 기능을 개발한 이후 서비스를 배포하기 위한 웹 시스템을 구축하는 강의입니다.

웹 시스템에 대한 기초 지식(HTML, CSS, JavaScript, jQuery, Bootstrap, Flask)을 활용하여 딥러닝 서비스를 배포하기 위한 웹 시스템 개발을 실습합니다.

## 딥러닝 서비스 배포 서버 구축
- Prerequsites (사전에 알아두면 좋은 지식)
    - HTML/CSS/JavaScript
    - Flask
    - jQuery
    - BootStrap
    - Python
